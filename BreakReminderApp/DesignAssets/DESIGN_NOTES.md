# 休息提醒器设计说明

## 色彩与风格说明

### 主色调
- **主色**: `#2D9CDB` (清新蓝) - 用于主要按钮、高亮元素
- **辅助色**: `#56CCF2` (浅蓝) - 用于次要强调、悬停状态
- **背景色**: `#F5F7FA` (极浅灰蓝) - 主窗口背景
- **纹理色**: `#FAFBFC` (几乎纯白) - 背景纹理底色

### 配色逻辑
- 采用 **蓝绿渐变** 作为图标主色，传递"清爽、专注、健康"的视觉感受
- 整体低饱和度设计，避免视觉疲劳，适合办公环境长时间使用
- 白色与浅灰为主，蓝色作为点缀，保持 macOS 原生应用风格

### 风格特征
- **极简主义**: 去除多余装饰，保留必要视觉元素
- **圆润友好**: 图标使用 58px 圆角，按钮建议 8px 圆角
- **层次分明**: 通过阴影和色彩明度区分层级，而非强烈对比

## 动效建议

### 1. 窗口出现（淡入 + 上移动）
- **时长**: 200-300ms
- **曲线**: `cubic-bezier(0.25, 0.46, 0.45, 0.94)` (ease-out-quad)
- **效果**: 
  - Opacity: 0 → 1
  - Transform: translateY(12px) → translateY(0)
- **说明**: 柔和出现，不突兀，给用户自然的视觉过渡

### 2. 卡片元素错峰出现
- **时长**: 单个元素 250ms
- **延迟**: 每个卡片间隔 40ms
- **曲线**: `cubic-bezier(0.34, 1.56, 0.64, 1)` (轻微弹性)
- **效果**: 设置页各选项卡依次从下方浮入
- **说明**: 引导视线从上到下，避免信息一次性涌入

### 3. 提醒弹窗（缩放 + 淡入）
- **时长**: 300ms
- **曲线**: `cubic-bezier(0.175, 0.885, 0.32, 1.275)` (ease-out-back)
- **效果**:
  - Opacity: 0 → 1
  - Transform: scale(0.92) → scale(1)
- **说明**: 弹窗需要引起注意，轻微回弹增加活力但不打扰

### 4. 按钮按下反馈
- **时长**: 100ms 按下, 200ms 回弹
- **曲线**: 
  - 按下: `ease-out`
  - 回弹: `cubic-bezier(0.34, 1.56, 0.64, 1)`
- **效果**: 
  - 按下: scale(0.96)
  - 回弹: scale(1.02) → scale(1)
- **说明**: 提供明确的触觉反馈，让用户感知操作被响应

### 5. 菜单栏图标状态切换
- **时长**: 150ms
- **效果**: 从正常状态到提醒状态时，添加轻微脉冲动画
- **说明**: 不打扰工作流但足够引起注意

## 可读性说明（为什么适合长时间办公）

### 1. 色彩对比度控制
- 文字与背景对比度维持在 **4.5:1 至 7:1** 之间（符合 WCAG AA 标准）
- 避免纯黑 (`#000000`) 与纯白 (`#FFFFFF`) 的极端对比，使用 `#1A1A1A` 与 `#F5F7FA`
- 减少蓝光伤害，背景采用偏暖的灰蓝色调

### 2. 字体选择
- **标题字体**: Inter Bold
  - 几何无衬线体，现代感强
  - 字母高度统一，小号下依然清晰
  - 适合短标题快速识别
  
- **正文字体**: Inter Regular
  - 与标题同属 Inter 字体家族，视觉统一
  - 优化的字间距，屏幕阅读友好
  - 数字等高宽，表格对齐整齐

### 3. 视觉层次
- 重要信息（如倒计时）使用 **大号字体 + 主色强调**
- 次要信息（如说明文字）使用 **小号 + 灰色**
- 间距充足：元素间留白大于内容本身，减少视觉压力

### 4. 减少视觉噪音
- 背景纹理使用 **极浅色调点阵**，密度低，不干扰前景阅读
- 图标采用 **扁平化 + 微渐变**，避免拟物化的复杂细节
- 窗口无边框或极细边框，融入桌面环境

### 5. 护眼设计
- 避免大面积高饱和度色块
- 提醒弹窗使用柔和阴影而非刺眼边框
- 支持跟随系统深色模式（预留适配接口）

## 设计原则总结

> **清爽而不冷淡，专注而不压抑**

休息提醒器的核心任务是保护用户健康，设计应当：
1. 在需要时出现，不打扰工作流
2. 视觉温和，不制造焦虑
3. 操作反馈明确，减少认知负担
4. 长时间使用不疲劳，符合人体工学
