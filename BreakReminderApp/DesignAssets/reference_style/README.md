# 风格参考文件夹

## 说明

此文件夹存放**3张艺术插画风格图**，用于指导应用的UI设计风格。

这些图片定义了你想要的"Neo-Noir 新黑色电影"视觉风格。

## 图片清单（请按顺序放入）

| 文件名 | 风格名称 | 核心特点 | 设计应用 |
|--------|----------|----------|----------|
| style_ref_01_gia.jpg | GIA鸡尾酒 | 黑底+橙色高光+几何分割+宽字间距 | 按钮样式、字体排版 |
| style_ref_02_neon.jpg | 荧光绿几何 | 黑底+荧光绿+对角线分割 | 工作模式配色、强调色 |
| style_ref_03_artdeco.jpg | Art Deco吸血鬼 | 酒红底+装饰边框+飞鸟图案 | 弹窗卡片、装饰元素 |

## 风格关键词

- **Neo-Noir**: 深色背景 + 高对比亮色点缀
- **Art Deco**: 几何对称 + 装饰边框 + 精致细节
- **Illustration**: 扁平化 + 线条感 + 色块分明
- **Retro-Futurism**: 复古未来感

## 设计要素提取

### 从 GIA 风格提取
- 主按钮：左侧色条 + 发光效果
- 字体：大写 + 宽字间距 (0.15-0.3em)
- 配色：纯黑底 + 橙色强调

### 从荧光绿风格提取
- 强调色：霓虹发光效果
- 进度指示：发光描边
- 开关：荧光色滑块

### 从 Art Deco 提取
- 卡片边框：双层 + 角落装饰
- 分割线：渐变 + 中间宽两头窄
- 标题：居中 + 宽字间距 + 发光

## 使用方法

1. 将你发的3张风格参考截图保存到此文件夹，按上表命名
2. 连同 reference_bg/ 的12张图一起发给设计AI
3. 设计AI会基于这些参考重新设计应用风格
