# Contributing

## Branching

- Main branch: `main`
- Feature branch prefix: `codex/`

## Commit style

- `feat:` new feature
- `fix:` bug fix
- `chore:` maintenance

## Local checks

```bash
swift test
```

## Release

- Tag format: `vMAJOR.MINOR.PATCH`
- Release pipeline: `.github/workflows/release.yml`
